# tk-note

使用Electron、React、TypeScript编写的笔记软件

## 软件界面

![](assets/img.png)

## 项目初始化

### 安装

```bash
$ pnpm install
```

### 开发

```bash
$ pnpm dev
```

### 编译

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
